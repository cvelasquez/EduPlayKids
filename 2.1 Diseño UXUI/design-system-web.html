📋 Guías de Uso

Principios y mejores prácticas para implementar el sistema de diseño de EduPlayKids.

✅ Hacer

  • Usar botones mínimo 48px para niños
  • Mantener contraste mínimo 7:1
  • Incluir feedback auditivo
  • Proporcionar ARIA labels
  • Usar animaciones suaves
  • Validar con niños reales

❌ No Hacer

  • Usar elementos menores a 44px
  • Sobrecargar con información
  • Animaciones muy rápidas
  • Colores de bajo contraste
  • Ignorar la accesibilidad
  • Textos muy largos
