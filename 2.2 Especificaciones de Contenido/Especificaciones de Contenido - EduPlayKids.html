Progresión por Edad

Pre-K (3-4 años) - Exploración y Descubrimiento

Enfoque: Exploración sensorial, juego libre, conceptos básicos

Duración: 3-5 minutos por actividad

Interacción: Touch directo, arrastrar simple

Kindergarten (5 años) - Reconocimiento y Patrones

Enfoque: Reconocimiento, patrones simples, seguimiento de instrucciones

Duración: 5-8 minutos por actividad

Interacción: Secuencias cortas, selección múltiple

Grados 1-2 (6-8 años) - Aplicación y Síntesis

Enfoque: Aplicación, resolución de problemas, pensamiento crítico

Duración: 8-12 minutos por actividad

Interacción: Múltiples interacciones, construcción

Sistema de Feedback y Retroalimentación

✅ Feedback Positivo (Respuesta Correcta)

  • Visual: Sparkles, checkmark con bounce, progreso visible
  • Audio: Chime agradable, elogios vocales, celebración de mascota
  • Háptico: Vibración suave de éxito (150ms)

🔄 Feedback Correctivo (Respuesta Incorrecta)

  • Visual: Shake suave, highlight rojo suave, pista revelada
  • Audio: Buzz suave no punitivo, aliento motivador
  • Educativo: Explicación después de 3 intentos
// Ejemplo: Configuración de feedback feedback_correct: { visual: "sparkles_burst", audio: "success_chime.mp3", voice: ["¡Excelente!", "¡Muy bien!"], haptic: "success_pattern_150ms" } feedback_incorrect: { visual: "gentle_shake", audio: "soft_buzz.mp3", voice: ["Intenta otra vez", "Casi lo tienes"], hint_after: 3 }

Especificaciones por Tipo de Asset

🖼️ Imágenes

  • Formato primario: PNG para transparencia
  • Formato secundario: WEBP para fotografías
  • Resolución base: xxxhdpi (4x) → downscale
  • Compresión: 60-70% reducción tamaño
  • Tamaño máximo: 500KB por imagen

🎵 Audio

  • Formato: MP3, 128 kbps, 44.1 kHz
  • Voces: Mono, normalizadas LUFS -23
  • Música: Estéreo, normalizadas LUFS -16
  • Efectos: 96 kbps, mono
  • Locutores: Nativos español/inglés

🎬 Animaciones

  • Formato principal: Lottie (JSON)
  • Frame rate: 30 fps
  • Duración: 2-10 segundos típico
  • Tamaño: <500KB animaciones complejas
  • Optimización: Vectorial escalable

🎞️ Video

  • Uso limitado: Solo casos justificados
  • Formato: MP4 (H.264)
  • Resolución: 720p máximo
  • Duración: 30-60 segundos máximo
  • Bitrate: 1-2 Mbps máximo
